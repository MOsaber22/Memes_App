# Meme Viewer Web App 😂

A simple web application that fetches memes from the Imgflip API and displays a meme based on the user’s input index.

## 🚀 Features
- Fetch memes from a public API
- Display meme name and image
- Input validation
- Error handling for invalid numbers

## 🛠 Technologies Used
- HTML
- CSS (Bootstrap)
- JavaScript (ES6)
- Fetch API

## 📦 How It Works
1. User enters a number between 0 and 99.
2. The app fetches meme data from the API.
3. The meme at that index is displayed.
4. If the number is invalid, an error message appears.

## 🌐 API Used
https://api.imgflip.com/get_memes
